# Astra Frontend

React tabanlı frontend uygulaması.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Uygulama http://localhost:3000 adresinde çalışacaktır.

## Build

```bash
npm run build
```

## Özellikler

- Kullanıcı kayıt ve giriş
- JWT token yönetimi
- Profil güncelleme
- Ürün listeleme
- Responsive tasarım
